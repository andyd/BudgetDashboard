# Component Examples

This directory contains example usage files for various components in the Budget Dashboard application.

These `.example.tsx` files demonstrate how to use specific components and are intended for documentation and reference purposes only. They are not part of the application build.

## Files

- `AdminDataTable.example.tsx` - Admin data table usage examples
- `BudgetLevelIndicator.example.tsx` - Budget level indicator examples
- `BudgetTooltip.example.tsx` - Budget tooltip examples
- `BudgetTreemap.example.tsx` - Treemap visualization examples
- `ComparisonPresets.example.tsx` - Comparison presets examples
- `ComparisonResult.example.tsx` - Comparison result display examples
- `DataFreshnessIndicator.example.tsx` - Data freshness indicator examples
- `DrillDownPanel.example.tsx` - Drill-down panel examples
- `empty-states.example.tsx` - Empty state component examples
- `ShareModal.example.tsx` - Share modal examples
- `SourceCitation.example.tsx` - Source citation examples
- `SpotlightPanel.example.tsx` - Spotlight panel examples
- `UnitSelector.example.tsx` - Unit selector examples

## Usage

These files are for reference only. To see components in action, refer to their actual implementations in `src/components/`.

## Note

These files are excluded from the main application build and ESLint checks by being located outside the `src/` directory.
